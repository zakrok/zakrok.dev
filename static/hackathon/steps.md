Instruction set of hackathon membership activation

1. Start by doing the entry of the hackathon
2. Make a direction and make a simple bash script with this:
```sh
touch ~/keyToHackathon.sh
echo "echo \"hello there! You are accepted in the challenge! :)\"" > ~/keyToHackathon.sh
chmod +x ~/keyToHackathon.sh
~/keyToHackathon.sh
```
